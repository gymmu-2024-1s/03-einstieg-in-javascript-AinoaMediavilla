Aufgaben für Bedingungen und Schleifen

Sortierverfahren

  1. bubble sort:

    bubble sort: es ist ein einfacher Sortieralgorithmus. Der wiederholtbenachbarte Elemente vergleicht und manchmal vertauscht er sie auch. Dieser Vorgang wird so lange wiederholt bis keine Vertauschungen mehr möglich/nötig ist, weil die Liste sortiert ist.

    2. sortier 2

    merge sort

    Merge Sort ist ein effizienter Teiler basierter Sortieralgorithmus der die Liste rekursiv in zwei Hälften teilt bis jede Teil Liste nur ein Element enthält. Dann werden die Teillisten sortiert zusammengeführt (so zu sagen gemerged), wobei die Elemente während des Zusammenfügens in der richtigen Reihenfolge angeordnet werden.

  2. sortier 3

    Quick sort

    Quick Sort: Es ist ein schneller Teiler basierter Sortieralgorithmus. Der ein Pivot-Element auswählt und das Array so aufteilt. Sodas alle Elemente kleiner als das Pivot links sind und alle größeren Elemente rechts davon stehen. Dieser Vorgang wird rekursiv auf die beiden Teil Listen angewendet bis das gesamte Array sortiert ist.
